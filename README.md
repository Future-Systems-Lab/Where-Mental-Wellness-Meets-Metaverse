# Where Mental Wellness Meets the Metaverse

End-to-end mental-wellness ecosystem for healthcare. Built with **analytics**, **blockchain**, and **governance** at the core.

## What It Delivers
- Identity & Consent: wallet-based auth (SIWE/EIP-712), DID consent flows
- Encrypted Health Data: private exchange and access control
- Program NFTs: L1/L2/L3 progression for verified milestones
- Incentive Token: ERC-20 engagement and retention
- Analytics: dashboards for participation, outcomes, and safety signals
- Governance: HIPAA-aligned privacy, auditability, policy controls

## Public Demos
- Secure Health Login (auth layer): https://secure-health-login.vercel.app  
- VITA Analytics (example dashboards): https://dune.com/dr_meg/vita-dashboard
